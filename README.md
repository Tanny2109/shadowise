# Introduction

Get shadows for any element.

#INSTALLATION   

`npm i shadowise --save`

#USAGE

```
import {shadowise} from 'shadowise';

shadowise ({
    shadow_type: 'soft',
    padding: false
});

```

#Options

Library supports following optional options:

* *shadow_type* - _hard | soft_ (Default: soft)
* *padding* - _boolean_ (Default is false)
